### 什么是TDCs？

时间数字转换器（TDCs）是一种用于测量时间间隔的设备，其测量范围通常在1纳秒到皮秒之间。TDCs最初是为科学研究目的而研发的，通过将时间模拟转换（TAC）和模拟数字转换（ADC）相结合来实现。

### TDCs的应用背景

尽管TDCs在科学研究中表现出色，但由于时间模拟转换（TAC）的复杂设计和测量频率测量容量的限制，这种技术并未广泛应用于工业领域。
